# SLAM Backend notes

BA即可以用来求解基于相邻帧的位姿估计问题，n个点，但只有一个位姿
也可以用来求基于时间序列的多个位姿和多个点的联合优化问题

场景一：基于相邻帧的位姿估计 - 优化变量少，通常只优化一个位姿，而三维点坐标已知
场景二：时序联合优化 - 优化变量多，同时优化多个位姿和多个三维点的坐标，可以用于SLAM的局部的多帧和多地图点的联合优化
这两种场景本质上都是在最小化同一个物理模型——重投影误差（Reprojection Error），但其优化目标和问题规模有着本质的区别

重投影误差的定义基于欧式空间的最小二乘算法，可以基于扰动模型（使用一阶泰勒展开）和基于高斯牛顿或LM等非线性优化算法进行建模求解
而下一章节的位姿图优化问题为基于李代数空间的最小二乘的算法为在李代数空间定义，需要基于李代数的伴随表示的特性，以及一阶泰勒展开和BCH近似以及高斯牛顿或LM去求解
