# SLAM

- 传感器部分
    - 激光点云预处理（运动补偿）
    - 相机与图像，成像模型，小孔成像及畸变
    - imu和gps
- 三维刚体运动
    - 四元数
        - 表示三维旋转的状态
    - 旋转向量
        - 罗德里格斯公式和指数隐射的泰勒展开的关联
    - 李群李代数
        - BCH公式的指数对数运算和李括号公式
        - 李代数的导数
        - 李代数的扰动模型
- 非线性优化方法
    - 基于贝叶斯原理的最大后验估计
    - 高斯牛顿方法，基于李代数的扰动模型迭代求解
    - 列文伯格—马夸尔特方法，更加稳定的迭代算法
- 视觉里程计
    - two view geometry初始化相机位姿，在此基础上的三角化算法求解深度
    - PnP（基于成像模型求解R，t）以及P3P(基于三角相似性)和RANSC随机一致性算法
    - Bundle Ajustment基于非线性最小二乘，基于李代数的高斯牛顿迭代法求解位姿
    - ICP基于SVD的多次迭代求解R，t
- 后端
    - 后端1：卡尔曼滤波，EKF，全局BA，ESKF
    - 后端2：pose graph，位姿图优化，因子图优化等
- 回环检测
    - 图像中基于bow的方法
    - cartographer中采用的匹配搜素的方式（基于激光点云）

- 参考实验：https://github.com/gaoxiang12/slambook
